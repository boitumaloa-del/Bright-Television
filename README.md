üîç Objectives

This project aims to:

- Clean and transform viewership data using Snowflake SQL.
- Analyze patterns in viewing time, frequency, and channel consumption.
- Identify peak viewing hours, daily patterns, and low-consumption days.
- Segment users by age, gender, province, and engagement level.
- Evaluate channel portfolio performance and genre-level engagement.
- Assess retention, user progression, and heavy-user degradation.
- Generate visuals and insights to inform content strategy and scheduling.
